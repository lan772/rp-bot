# RP-бот: пошаговая инструкция

## Шаг 1. Создать бота в Discord

1. Зайди на https://discord.com/developers/applications
2. Нажми **New Application**, дай название (например, RP System)
3. Слева перейди в раздел **Bot** → **Add Bot**
4. Включи переключатель **MESSAGE CONTENT INTENT** (обязательно, иначе команды не будут работать)
5. Нажми **Reset Token** → скопируй токен (это секретный ключ бота, никому не показывай)
6. Слева в разделе **OAuth2 → URL Generator**:
   - в Scopes выбери `bot`
   - в Bot Permissions выбери: Send Messages, Read Message History, Embed Links, Manage Messages
   - скопируй сгенерированную ссылку внизу и перейди по ней, чтобы добавить бота на свой сервер

## Шаг 2. Узнать ID роли ГМ (опционально)

Если хочешь, чтобы редактировать статы могли не только админы, а отдельная роль "ГМ":
1. В Discord включи режим разработчика: Настройки → Расширенные → Режим разработчика
2. Правой кнопкой по роли ГМ на сервере → Копировать ID
3. Впиши этот ID в файл `bot.py` вместо `GM_ROLE_ID = None`, например: `GM_ROLE_ID = 123456789012345678`

Если оставить `None` — редактировать смогут только участники с правами администратора сервера.

## Шаг 3. Разместить бота на Railway (бесплатный хостинг)

1. Зайди на https://railway.app и зарегистрируйся (можно через GitHub)
2. Нажми **New Project** → **Empty Project**
3. Нажми **+ New** → **GitHub Repo** (если файлы бота в репозитории) ИЛИ **Empty Service** и вручную загрузи файлы через вкладку Deploy
   - Самый простой способ: создай новый репозиторий на GitHub, загрузи туда `bot.py`, `requirements.txt`, `Procfile`, затем в Railway выбери **Deploy from GitHub repo**
4. После создания сервиса зайди в **Variables** и добавь переменную:
   - Ключ: `DISCORD_TOKEN`
   - Значение: токен бота из Шага 1
5. Railway автоматически определит `Procfile` и запустит бота как worker-процесс
6. В логах (**Deployments → View Logs**) должно появиться "Бот запущен как [имя]"

## Слоты персонажей

У каждого игрока теперь 6 слотов (номера 1-6) — можно вести до 6 разных персонажей на одном аккаунте. В большинстве команд номер слота указывается сразу после команды/после игрока.

## Команды бота

**Просмотр (доступно всем):**
- `!информация` — слот 1, своя карточка
- `!информация 3` — слот 3, своя карточка
- `!информация 2 @игрок` — слот 2 другого игрока
- `!слоты` — список всех 6 слотов и имена персонажей в них (свои)
- `!слоты @игрок` — то же самое для другого игрока

**Редактирование (только ГМ/админ) — везде первым указывается игрок, вторым номер слота:**
- `!добавить_стат @игрок 1 сила 1`
- `!убрать_стат @игрок 1 сила 1`
- `!добавить_трейт @игрок 1 Хищный инстинкт`
- `!убрать_трейт @игрок 1 Хищный инстинкт`
- `!добавить_артефакт @игрок 1 Клинок рассвета`
- `!убрать_артефакт @игрок 1 Клинок рассвета`
- `!достижение @игрок 1 Победил первого босса`
- `!уровень @игрок 1 5`
- `!имя @игрок 1 Дракон Пепла`
- `!фото @игрок 1` — прикрепи изображение к этому же сообщению (просто вставь картинку в тот же чат, где пишешь команду)

Доступные статы для команд: прочность, сила, скорость, реакция, стойкость, регенерация, контроль_маны, энергия

Финальный список статов, которые показывает бот: **Прочность, Сила, Скорость, Реакция, Стойкость, Регенерация, Контроль маны, Мана/Энергия**
